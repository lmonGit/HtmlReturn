Chapitre 1 - Comment créer un site Web

Section 1 - Le fonctionnement d'un site Web

Avant de créer un site Web, il faut savoir comment en consulter un. Pour cela tout le monde sait aujourd'hui utiliser un navigateur. Un navigateur ou "browser" est un programme qui sert à consulter les milliards de site qui existent aujourd'hui. Ce sont : Google Chrome, Mozilla Firefox ou encore Opéra.
Le Web a été développé par Tim Berners Lee en 1991. Le Web ne représente qu'une partie d'Internet. Internet c'est en fait : le Web, les mails, les newsgroups...

Les sites consultés tous les jours sont le fruit de deux langages qui sont les fondations de tous les sites Web. Ces deux langages sont le langage HTML et le langage CSS.

Section 2 - Les rôles de HTML et CSS

HTML

Son rôle est de gérer et organiser le contenu. C'est donc en HTML que vous écrirez ce qui doit être affiché sur la page : du texte, des liens, des images... Il y a eu 5 versions de HTML entre 1991 et 2017. Le HTML est un langage de balisage.

CSS

Le rôle du CSS est de gérer l'apparence de la page web (agencement, positionnement, décoration, couleurs, taille du texte... Il y eu 3 versions de CSS entre 1996 et 2017

Section 3 - Choisir un éditeur de texte

On peut classer ces logiciels de création de site web en deux catégories

Sous-section 1 - Les WYSIWYG

(What You See Is What You Get - Ce Que Vous Voyez Est Ce Que Vous Obtenez) : ce sont des programmes qui se veulent très faciles d'emploi, ils permettent de créer des sites web sans apprendre de langage particulier. Parmi les plus connus d'entre eux : Mozilla Composer, Microsoft Expression Web, Dreamweaver...

Sous-section 2 - Les éditeurs de texte

Ce sont des programmes dédiés à l'écriture de code. On peut en général les utiliser pour de multiples langages, pas seulement HTML et CSS.

Sublime Text

Sublime Text est un éditeur de texte devenu très populaire parmi les développeurs. On l'utilise aussi bien pour développer en HTML et CSS que dans d'autres langages (Python, Ruby, etc.). Il fonctionne sur Windows, Mac OS X et Linux.Il a l'avantage d'être simple, épuré et facile à lire dès le départ. Pas de centaines de boutons dont on ne comprend pas à quoi ils servent.Notepad ++ est aussi intéressant pour sa simplicité

Aptana Studio était surtout tourné vers les utilisateurs de Mac, mais il a été porté à windows. Aptana semble être un compromis entre Sublime text et Notepad ++. Un trés bon outil pour moi.

Les environnements de développement intégré

Netbeans est un IDE de référence soutenu par Oracle. Sa complexité relative cache en fait des possibilités plus étendues, notamment pour l'intégration de code d'autres langages. L'idéal est d'utiliser Netbeans et Aptana Studio.

Section 4 - Les navigateurs

Il existe de nombreux navigateurs différents. Voici les principaux à connaître.
Google Chrome : le navigateur de Google, simple d'emploi et très rapide. C'est le navigateur que j'utilise au quotidien. Compatibilité : Windows, Mac et Linux
Mozilla Firefox : le navigateur de la fondation Mozilla, célèbre et réputé. Je l'utilise fréquemment pour tester mes sites web. Compatibilité : Windows, Mac et Linux
Internet Explorer : le navigateur de Microsoft, qui équipe tous les PC Windows. Je l'utilise fréquemment pour tester mes sites web. Compatibilité : Windows
Safari : le navigateur d'Apple, qui équipe tous les Mac. Compatibilité : Windows et Mac
Opera : l'éternel outsider. Il est moins utilisé mais propose de nombreuses fonctionnalités. Compatibilité : Windows, Mac et Linux

Il est conseillé d'installer plusieurs navigateurs sur son ordinateur pour s'assurer que son site fonctionne correctement sur chacun d'eux Les navigateurs ne connaissent pas toujours les dernières fonctionnalités de HTML et CSS.Des sites comme normansblog.de et caniuse.com tiennent notamment à jour une liste des fonctionnalités CSS prises en charge par les différentes versions de chaque navigateur.Le célèbre site caniuse.com tient à jour une liste des fonctionnalités prises en charge par les différentes versions de chaque navigateur.Il existe des navigateurs pour les mobiles.
