Chapitre 7 - Les tableaux

!!! AVERTISSEMENT !!! Il sera plus profitable de n'introduire dans ce pseudo-site sur HTML que le code d'une application des notions tirées du livre de Mathieu Nebra car reprendre intégralement son cours pourrait introduire des erreurs. Dorénavant, je ne présenterai que le code servant d'application des notions présentées. Le code sera commenté de manière précise mais il ne faudra pas oublier que si la fourniture de mon code avait lieu un jour, les commentaires ne sont pas nécessaires. Les commentaires permettront d'ancrer en situation réelle les notions abordées.

Section 2 - Un tableau structuré

Pour écrire un tableau structuré il faut écrire les balises dans cet ordre

  1. ‹ thead ›
  2. ‹ tfoot ›
  3. ‹ tbody ›
